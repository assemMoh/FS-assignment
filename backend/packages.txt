1- create virtual environment
    => py -3 -m venv <name of environment>

2- activate the environment
    => <name of environment>\Scripts\activate

3- install flask
    => pip install flask

4- install dotenv for environment variables 
    => pip install python-dotenv

5- install psycopg2 to with database
    => pip install psycopg2-binary

6- install sqlalchemy library for using ORM
    => pip install flask-sqlalchemy

7- install mysqldb for using mysql as database
    => pip install flask-mysqldb
    => pip install mysql-connector
    => pip install mysql-connector-python
    => pip install mysql-connector-python-rf
    => pip install pymysql
    => pip install cryptography

8- install bcrypt to encrypt password
    => pip install bcrypt

9- install login to handle logged in user and protect apis 
    => pip install login
